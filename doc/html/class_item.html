Detailed Description

Clase abstracta padre las subclases: Juguete, Comida y Medicina.

Esta clase posee todos los métodos disponibles para cada subclase de Item. Lógica tras la ejecución de funciones miembro (excepto SETers y GETers) varían según sub-clase.

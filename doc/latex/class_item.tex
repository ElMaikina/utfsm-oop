\doxysubsection{Detailed Description}
Clase abstracta padre las subclases\+: \doxylink{class_juguete}{Juguete}, \doxylink{class_comida}{Comida} y \doxylink{class_medicina}{Medicina}. 

Esta clase posee todos los métodos disponibles para cada subclase de \doxylink{class_item}{Item}. Lógica tras la ejecución de funciones miembro (excepto SETers y GETers) varían según sub-\/clase.

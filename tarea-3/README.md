## Ejecución

Ejecutar _make_ desde la terminal de su preferencia en la etapa especifica, esto compilará el programa. Posterior a eso, ejecutar el
archivo binario generado. Por ejemplo, para la etapa dos:

```
cd etapa-2/
make
./App
```
